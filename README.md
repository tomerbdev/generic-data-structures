# Generic Data Structures

A collection of basic data structures implemented in both Java and C#.

## Contents
- **generic-one-way-linked-list-node**: Basic building block for stacks/queues.
- **generic-two-way-linked-list-node**: Doubly linked list node.
- **generic-bin-tree**: Binary tree node.
- **generic-stack**: LIFO structure.
- **generic-queue**: FIFO structure.

## Usage
These are fundamental implementations meant for educational purposes to demonstrate how node-based data structures work.
